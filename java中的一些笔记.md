## java

### java中的引用类型

- 强引用

日常写的任何类都是强引用类型

- 软引用

软的引用，当jvm内存不够的时候被回收

- 弱引用

被垃圾回收期发现就会被回收

- 虚引用



## Commons BeanUtils 和 Spring BeanUtils 的区别

1、copyProperties 的 dest 和 source 是相反的位置
2、Spring BeanUtils 可以指定忽略的字段以及可以指定目标类
